Javascript Snake Game

Instructions:

Welcome to the Javascript Snake Game created by Ty Petty!
The instructions for the game are as follows:

  1. Press any of the arrow keys to start the game and change directions.
  2. Move the green snake to eat the white square.
  3. Do not hit the edges of the game board.
  4. See your score go up as you progress!
Thanks for playing! I appreciate you enjoying the game I created!
